> Point your camera at ingredients. Get instant recipe recommendations powered by computer vision.

Smart Chef uses **real-time video analysis** to identify ingredients through your webcam, then recommends matching recipes using a custom mathematics-based scoring algorithm.

Designed as an elite modern application bridging the gap between Pre-Trained Machine Learning and Custom Computer Vision Algorithms, all running locally.

---

## ✨ Features

- **🎥 Real-time Detection** — WebSocket-powered live local video analysis at ~2fps.
- **🧠 Hybrid AI Pipeline** — A custom trained YOLOv8 object detection model paired directly with custom classification logic (HSV, LBP, Color Histograms) to mathematically validate ingredients.
- **🍽️ Smart Recommendation** — Jaccard similarity + weighted coverage scoring algorithm.
- **📊 Interactive Nutrition Dashboard** — Automatic calorie, protein, carbs, and fat breakdown per ingredient.
- **🌙 Premium Dark UI** — Glassmorphism, particle animations, dynamic chips, recipe modals with true-to-life images and YouTube video tutorials.
- **🐳 Docker Containerized** — One command to run everything seamlessly.

## 🚀 Quick Start

### With Docker (Recommended)

```bash
docker-compose up -d --build
```

Then open [http://localhost](http://localhost) in your browser.

### Manual Setup (Without Docker)

**1. Start the Backend:**
```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8000
```

**2. Start the Frontend:**
```bash
cd frontend
npm install
npm run dev
```
Then open `http://localhost:5173/` in your browser.

## 🧠 Behind the Scenes: The Smart Chef Vision Pipeline

### 1. Object Detection (Elite Trained YOLOv8)
A specifically fine-tuned YOLOv8 model (`smartchef_custom_model.pt`) identifies bounding box areas of food objects in the video frame instantly.

### 2. Custom Classification & Mathematical Validation (No ML Libraries)
Each bounding box generated by YOLO is processed through a strict **hand-written** mathematical validation pipeline using OpenCV:

| Step | Method | What It Does |
|------|--------|-------------|
| **Color Segmentation** | HSV masking | Identifies dominant underlying color signatures. |
| **Texture Analysis** | Local Binary Patterns (LBP) | Computes texture maps and histogram distances. |
| **Color Matching** | Chi-squared histogram | Evaluates correlation against a pure reference profile map. |
| **Shape Analysis** | Aspect ratio mechanics | Filters logically (e.g. distinguishing elongated carrots from round tomatoes). |

Final Confidence Score Formula: `(YOLO_Confidence * 0.92) + (Custom_OpenCV_Score * 0.08)`.
This handles extremely similar visually appearing ingredients without hallucination.

### 3. Recipe Matching Algorithms
A strictly algebraic implementation running via:
- **Jaccard Similarity Index**: `J(A,B) = |A∩B| / |A∪B|`
- **Ingredient Coverage Score**: Scaled by predefined importance.
- **Complexity Veto**: Recipes missing critical mass ingredients are deprioritized.
